# CCNA 2 SRWE Revisions

Site de révision interactif pour CCNA 2 SRWE.

## Accès

Lien GitHub Pages prévu : https://hervensson.github.io/ccna2-srwe-revisions/

Si le lien n'est pas encore actif, active GitHub Pages dans :

`Settings` → `Pages` → `Deploy from a branch` → branche `main` → dossier `/root` → `Save`.

## Contenu

- 171 blocs de questions extraits du document fourni
- 70 questions aléatoires par session normale
- timer global de 1h15
- correction finale uniquement
- questions difficiles, historique des scores, thèmes CCNA
- mode refaire uniquement les erreurs
- questions d'association interactives quand les paires sont fiables
