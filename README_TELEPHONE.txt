CCNA 2 SRWE - Site de revision

Contenu du pack :
- index.html : page a ouvrir
- styles.css, app.js, questions.js : fichiers necessaires au quiz
- assets/ : illustrations extraites du document Word

Utilisation sur ordinateur :
1. Dezippe le fichier ccna2-srwe-revisions-site.zip.
2. Ouvre index.html avec Chrome, Edge, Firefox ou Safari.

Utilisation sur telephone :
1. Envoie le fichier zip sur ton telephone.
2. Dezippe-le avec l'application Fichiers / Files.
3. Ouvre index.html dans ton navigateur.

Important :
- Le site fonctionne hors ligne.
- La sauvegarde de progression utilise le stockage du navigateur.
- Sur certains telephones, si index.html est ouvert depuis l'application Fichiers, la sauvegarde peut etre limitee. Dans ce cas, garde le meme navigateur et ne deplace pas le dossier apres avoir commence une session.

Banque de questions :
- 171 blocs de questions ont ete extraits du document source.
- 157 questions sont notees automatiquement.
- 14 questions d'association ou questions sans choix fiable sont conservees en mode correction manuelle.
- Le document source annonce une numerotation jusqu'a 174, mais il contient des numeros manquants et des numeros dupliques. Le site inclut tout ce qui est extractible proprement du fichier fourni, sans inventer les questions absentes.
